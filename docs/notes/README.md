# Notes

Obsidian で増やしていく作業ノートの入口。

## 入口

- [Inbox](./inbox/README.md)
- [Daily](./daily/README.md)
- [Investigations](./investigations/README.md)
- [Decisions](./decisions/README.md)
- [MVP Architecture](../mvp-architecture.md)
- [Coding Rules](../coding-rules.md)
- [OpenSpec](../../openspec/README.md)

## 置き場所

- 一時メモの起点: `docs/notes/inbox/`
- 日次ログ: `docs/notes/daily/`
- 調査メモ: `docs/notes/investigations/`
- 意思決定メモ: `docs/notes/decisions/`
- 添付ファイル: `docs/assets/`
- テンプレート: `docs/templates/`

## 運用ルール

- まず `inbox/` や `daily/` に書き、残すべき内容だけ整理して昇格する
- 永続的に残す設計判断は `docs/` または `openspec/` に反映する
- 一時的な調査ログや作業メモはこの配下に置く
- GitHub でも読みやすいように Markdown リンクを使う

## 作成例

- `inbox/2026-03-15-obsidian-setup.md`
- `investigations/rendering-investigation.md`
- `decisions/preview-rendering-direction.md`
