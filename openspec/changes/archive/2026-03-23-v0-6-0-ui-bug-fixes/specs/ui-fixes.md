# v0.6.0 UI Bug Fixes

## 概要

Katana UI の表示不良・パフォーマンスに直結する Critical/High 優先度のバグを修正する。

## 要件

以下のバグを修正し、各テスト（単体・結合）を 100% 通過させること。

- **[MUST: P0] CPU過剰利用（無限再帰描画）の修正**
  - KatanaSvgLoader が SVG 読み込みに失敗した際、エンドレスに `LoadError::Loading` を返し続けることで EgUI が 100% CPU で無限ループ再描画してしまう問題を解決する。

- **[MUST: P1] 起動時のメインUI先行表示とスプラッシュ非表示バグ**
  - 起動時フリーズを隠蔽するための記号ロード画面（スプラッシュ画面）が表示されず、未構築のメインUIが先行して表示されてしまう問題を治癒する。

- **[MUST: P3] Blockquote (引用) 描画時の上下不要余白の除去**
  - 引用（blockquote）の左枠線（`|`）が、テキスト本体の上下に必要な空行を含んで不自然に長く伸びてしまう描画バグを除去する。

- **[MUST: P4] コードブロックSVGアイコンの警告マーク（⚠️）表示バグ**
  - コピーアイコン実装の副作用により、本来は「空のコードブロック」の時のみ表示すべき警告アイコンが、内容がある場合にもフォールバック（読み込み失敗）扱いで表示されてしまう不具合を修正する。
