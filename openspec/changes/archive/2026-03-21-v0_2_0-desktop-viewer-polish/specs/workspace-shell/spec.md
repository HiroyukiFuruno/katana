## MODIFIED Requirements

### Requirement: ワークスペースパネル
既存のワークスペースパネルにコンテキストメニューと永続化機能を統合する。ツリーエントリの右クリック操作と、ワークスペース一覧の管理UIを追加する。

#### Scenario: ツリーエントリのコンテキストメニュー表示
- **WHEN** ユーザーがワークスペースツリーのエントリを右クリックする
- **THEN** エントリ種別（ファイル/フォルダ）に応じたコンテキストメニューが表示される

#### Scenario: ワークスペース一覧の管理
- **WHEN** 複数のワークスペースディレクトリが追加されている
- **THEN** ワークスペースパネルのヘッダーに一覧管理UIが表示され、各ワークスペースの切り替えと除去が可能

#### Scenario: ワークスペースの自動復元
- **WHEN** アプリを再起動する
- **THEN** 前回追加していたワークスペースディレクトリが自動的に復元され、ツリーが表示される
