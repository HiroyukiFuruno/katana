## ADDED Requirements

### Requirement: フォルダコンテキストメニューで全ファイルを開く
ワークスペースツリーのフォルダエントリを右クリックした際、コンテキストメニューに「全て開く」のオプションを表示する。選択時、そのフォルダ直下のMarkdownファイルを全てタブとして開く。

#### Scenario: フォルダ内の全Markdownファイルを開く
- **WHEN** ユーザーがワークスペースツリーのフォルダを右クリックし「全て開く」を選択する
- **THEN** そのフォルダ直下のMarkdownファイルが全て新しいタブとして開かれる

#### Scenario: サブフォルダのファイルは開かない
- **WHEN** ユーザーが「全て開く」を実行したフォルダにサブフォルダがある
- **THEN** サブフォルダ内のファイルは開かれない（直下のファイルのみ）

#### Scenario: 既に開いているファイルは重複して開かない
- **WHEN** 「全て開く」実行時にフォルダ内のファイルが既にタブで開かれている
- **THEN** 既存のタブはそのまま維持され、未開封のファイルのみが新しいタブとして追加される

### Requirement: ファイルメタ情報表示
ワークスペースツリーのファイルエントリを右クリックした際、コンテキストメニューに「メタ情報」のオプションを表示する。選択時、ファイルのメタ情報（サイズ、更新日時、パス）をダイアログで表示する。

#### Scenario: ファイルメタ情報の表示
- **WHEN** ユーザーがファイルを右クリックし「メタ情報」を選択する
- **THEN** ファイルサイズ、最終更新日時、絶対パスを含むダイアログが表示される

#### Scenario: ディレクトリのメタ情報
- **WHEN** ユーザーがディレクトリを右クリックし「メタ情報」を選択する
- **THEN** ディレクトリの絶対パスと、直下のファイル数/サブディレクトリ数が表示される
