## ADDED Requirements

### Requirement: ワークスペース読み込み時のローディング表示
ワークスペースディレクトリを初回オープンまたは追加した際に、ファイルツリー構築中であることを示すローディングインジケーターを表示する。

#### Scenario: 初回ワークスペースオープン時のローディング
- **WHEN** ユーザーがワークスペースディレクトリを初めて開く
- **THEN** ワークスペースパネルにスピナーアイコンと「読み込み中...」のメッセージが表示される

#### Scenario: ワークスペース追加時のローディング
- **WHEN** ユーザーが2つ目以降のワークスペースディレクトリを追加する
- **THEN** 追加中のワークスペースエントリにスピナーアイコンが表示される

#### Scenario: 読み込み完了後のローディング解除
- **WHEN** ファイルツリーの構築が完了する
- **THEN** スピナーが消え、ファイルツリーが表示される

#### Scenario: 大規模ディレクトリでのUI応答性維持
- **WHEN** 数千ファイルを含むディレクトリを開く
- **THEN** ローディング中もUIがフリーズせず、他の操作（既存タブの閲覧等）が可能

#### Scenario: アプリ再起動時の永続化ワークスペース復元
- **WHEN** アプリ起動時に永続化されたワークスペースを復元する
- **THEN** 各ワークスペースにスピナーが表示され、読み込み完了後にツリーが表示される
