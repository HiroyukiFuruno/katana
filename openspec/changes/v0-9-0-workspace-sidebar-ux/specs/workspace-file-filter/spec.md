## MODIFIED Requirements

### Requirement: ツリーのフィルタートグルと入力UI

サイドバーのワークスペース操作ヘッダー部分に、フィルターモードのトグルボタンを配置しなければならない（MUST）。フィルタートグルは更新ボタンと同じ先頭側グループに保持し、トグルボタンを有効にすると、その直下に正規表現を入力するためのテキストフィールドが出現しなければならない（MUST）。

#### Scenario: フィルター入力UIの表示と非表示

- **WHEN** ユーザがワークスペースのフィルターボタンをクリックする
- **THEN** 直下にテキスト入力フィールドがドロップダウン表示される
- **THEN** 更新ボタンは同じ先頭側グループに表示されたまま維持される

#### Scenario: ヘッダー再配置後もフィルタートグルを利用できる

- **WHEN** ユーザーがワークスペースペインの操作ヘッダーを見る
- **THEN** フィルターボタンは更新ボタンと同じ先頭側グループに表示される
- **THEN** 全展開・全閉ボタンとは別のグループで区別される
