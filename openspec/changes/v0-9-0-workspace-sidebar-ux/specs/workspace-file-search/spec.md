## MODIFIED Requirements

### Requirement: ファイル名検索モーダルの表示
ユーザが任意のショートカットキーまたは左アクティビティレールの検索ボタンを介して呼び出したとき、ワークスペース全体からファイルを検索するためのモーダル（ポップアップウィンドウ）を表示しなければならない（MUST）。

#### Scenario: ショートカットで検索モーダルを開く
- **WHEN** ユーザーがワークスペース検索用のアクションを実行する
- **THEN** 中央に検索入力を受け付けるモーダルウィンドウが表示される

#### Scenario: アクティビティレールから検索モーダルを開く
- **WHEN** ユーザーが左アクティビティレールの検索ボタンを押す
- **THEN** 中央に検索入力を受け付けるモーダルウィンドウが表示される
- **THEN** ワークスペースペインが非表示でも検索操作を開始できる
